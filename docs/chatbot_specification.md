# 챗봇 기획서 (Chatbot Specification)

> 카카오톡 챗봇을 통한 주식 정보 제공
> 챗봇 = 요약/진입 채널, 웹 = 상세/시각화

---

## 공통 UI

### 초기 진입 화면

**노출 메시지:**
```
안녕하세요 :) 주토피아입니다 !
주식 투자에 필요한 정보를 도와드릴게요.
원하시는 기능을 선택해주세요.
```

**말풍선 + 큰 버튼 5개:**
1. 주식 계좌 연동
2. 종목 리포트
3. 주식 용어 사전
4. 거래내역 / 요약 리포트
5. 뉴스 / 커뮤니티

### 버튼 유형 구분

| 구분 | 말풍선 + 큰 버튼 | 퀵 버튼 |
|------|-----------------|--------|
| 역할 | 기능 "이동/진입" | 기능 "진행/선택" |
| 위치 | 기능 선택 화면(홈) | 기능 내부 대화 중 |
| 느낌 | 메뉴 버튼 | 선택지 버튼 |
| 결과 | 다른 블록/스킬로 이동 | 같은 블록 내 분기 |

# Chatbot_05. 뉴스 / 커뮤니티

## 0. 기능 목적

- 사용자가 관심 있는 종목에 대해 **현재 시장 분위기 (커뮤니티)** 와 **주가에 영향을 줄 수 있는 핵심 이슈 (뉴스)** 를 **짧고 이해하기 쉬운 형태로 제공**
- 챗봇은 정보의 "목록"이 아니라 **요약·해석·선택 유도** 역할을 수행

---

## 1. 기능 진입

**노출 메시지:**
```
종목에 대한 뉴스와 커뮤니티 정보를 제공해드려요 :)
궁금한 종목을 입력하시거나,
하단에 있는 퀵 버튼을 이용해 주세요!
```

**하단 퀵 버튼:**
1. 종목 직접 입력
2. 관심 종목
3. 보유 종목
4. 메인 화면으로

---

## 2. 계좌 연동 조건 처리

### 2-1. 계좌 미연동 시

**노출 메시지:**
```
해당 기능은 계좌 연결 후 이용할 수 있어요.
계좌를 연결해 주세요 :)
```

**하단 퀵 버튼:**
1. 계좌 연결하기
2. 종목 직접 입력
3. 이전 화면으로

### 2-2. 계좌 연동 시

- 사용자 관심 종목 / 보유 종목 리스트 조회

---

## 3. 종목 리스트 안내

### 3-1. 보유 종목

**노출 메시지:**
```
(사용자 이름)님의 보유 종목은
삼성전자, SK하이닉스 입니다.

어떤 종목에 대한 정보를 알려드릴까요?
```

### 3-2. 관심 종목

**노출 메시지:**
```
(사용자 이름)님의 관심 종목은
아마존 입니다.

어떤 종목이 궁금하신가요?
```

### 3-3. 종목 선택

- 사용자 입력으로 종목 선택

---

## 4. 정보 제공 구조

> 뉴스와 커뮤니티를 한 번에 섞지 않고 **'커뮤니티 → 뉴스'** 순서로 제공

---

## 5. 커뮤니티 정보 제공

### 5-1. 커뮤니티 요약 (1차 메시지)

**노출 메시지 구조:**
```
최근 (종목명)에 대한
커뮤니티 분위기부터 알려드릴게요

• 전반적으로 긍정적인 의견이 많아요
• 실적 개선 기대감이 주요 이유예요
```

**표현 규칙:**
- 감정 톤: 긍정 / 중립 / 부정 중 하나
- 실시간성 표현: "최근", "오늘 기준", "방금 전까지"

### 5-2. 대표 의견 요약 (2차 메시지)

**노출 메시지:**
```
대표적인 의견을 몇 개 보면 아래와 같아요 :)

- "실적 바닥은 지난 것 같다"
- "외국인 수급이 계속 유입 중"
- "단기 급등은 부담"

자세한 커뮤니티는 하단의 퀵 버튼을 눌러 웹에서 확인하세요 !
```

**규칙:**
- 실제 게시글 원문 미노출
- 요약 문장만 제공 (2~3개)

### 5-3. 커뮤니티 이후 퀵 버튼

1. 뉴스도 보기
2. 다른 종목 보기
3. 웹에서 자세히 보기
4. 기능 종료

---

## 6. 뉴스 정보 제공

### 6-1. 뉴스 요약 (1차 메시지)

**노출 메시지 구조:**
```
최근 (종목명) 관련
주요 뉴스도 정리해봤어요.

• 2분기 실적이 시장 예상치를 상회했어요
• 반도체 업황 회복 기대가 언급되고 있어요
```

**규칙:**
- 뉴스 개수: 최대 3~5건
- 제목 나열
- 사건/이벤트 중심 요약

### 6-2. 뉴스 필터링 기준 (내부 정책)

**1차 필터: 제목 키워드 기반**

| 카테고리 | 키워드 |
|---------|--------|
| 실적 | 실적, 매출, 영업이익, 순이익 |
| 주가 | 급등, 급락, 주가 |
| 주주환원 | 배당, 자사주, 증자 |
| 공시/규제 | 공시, 규제, 승인 |
| M&A | 인수, 합병, M&A |

**2차 필터: 투자 영향 가능성 분류**

| 등급 | 노출 여부 |
|------|----------|
| HIGH | O |
| MEDIUM | O |
| LOW | X |

### 6-3. 뉴스 이후 퀵 버튼

1. 기사 원문 보기
2. 다른 종목 보기
3. 웹에서 더 보기
4. 기능 종료

---

## 7. 실시간성 표현 정책

- 실제 실시간 피드 제공
- 언어적 표현으로 실시간성 전달

**사용 문구 예시:**
- "최근 몇 시간 기준으로"
- "오늘 커뮤니티에서 많이 언급되고 있어요"
- "방금 전 기준으로 이런 의견이 많아요"

---

## 8. 핵심 요약

| 항목 | 내용 |
|------|------|
| 챗봇 역할 | 목록이 아닌 **요약 채널** |
| 커뮤니티 | 분위기 (감정 톤) |
| 뉴스 | 사실 (이벤트/이슈) |
| 제공 개수 | 항상 3~5개 핵심만 |
| 상세 정보 | 웹으로 연결 |
